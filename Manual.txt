```
mkdir subscribers-app
cd subscribers-app
Create the package.json file
touch package.json
```
Copy this configuration into package.json
```
{
  "name": "subscribers-app",
  "scripts": {
    "dev": "nuxt",
    "build": "nuxt build",
    "generate": "nuxt generate",
    "start": "nuxt start"
  }
}
```
npm install 
npm run build
mkdir pages
touch pages/index.vue
```
To run Application
```
npm run dev
```
